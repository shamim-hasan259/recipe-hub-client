# 🍽️ RecipeHub - Recipe Sharing Platform

RecipeHub is a full-stack MERN application where food lovers can create, discover, purchase, and manage recipes. Users can publish their own recipes, save favorites, purchase premium recipes using Stripe, and report inappropriate content. Administrators can manage users, recipes, featured recipes, and reports through a dedicated dashboard.
